Instructions on the usage of the script:
1. Download the attachment and unzip it.
2. Put all the PDF files for which the first page is to be extracted in the unzipped folder "PDFs_first_page_extraction"
3. Double left-click on script "pdf_first_page_extraction_script.bat" present inside the folder.
4. The combined file "first_page_combined_pdf.pdf" containing the first page of all the PDFs will be formed in "first_page_of_pdf" folder present inside the main folder.
